# Pose Data

This directory contains pre-computed pose data JSON files generated by the Python preprocessing tools.

## Files

Each song should have a corresponding JSON file:
- `cheapthrills.json`
- `uptownfunk.json`
- `dontstartnow.json`
- `callmemaybe.json`
- `ghungroo.json`

## Generating Pose Data

Run the Python preprocessing tools from the `python-tools` directory:

```bash
cd python-tools
uv run python batch_process.py ../songs/ --output ../mobile/assets/poses/
```

## Format

Each JSON file contains:
```json
{
  "songId": "cheapthrills",
  "fps": 30.0,
  "totalFrames": 1500,
  "frames": [
    {
      "frameNumber": 0,
      "timestamp": 0.0,
      "keypoints": { ... },
      "angles": { ... }
    }
  ]
}
```
