## Usage

These audio files are played during gameplay, synchronized with the reference video. The app uses `expo-av` for audio playback.

## Synchronization

Audio is automatically synchronized with video playback. The app:
1. Loads both video and audio
2. Starts playback simultaneously
3. Maintains sync throughout gameplay
4. Handles pause/resume together
